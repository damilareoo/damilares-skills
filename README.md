# v-skills

A curated collection of Claude Code skills by [@damilareoo](https://github.com/damilareoo).

Covers design systems, Vue ecosystem, animation, typography, accessibility, frontend architecture, and more.

## Install

**Step 1 — Add this repo as a marketplace in Claude Code:**

```
/plugin marketplace add github:damilareoo/claude-skills
```

**Step 2 — Install the skill pack:**

```
/plugin install v-skills@damilareoo/claude-skills
```

Done. All skills are now available in your Claude Code sessions.

## Adding more skills

Drop a new `skills/<skill-name>/SKILL.md` into `plugins/v-skills/skills/` and push. Anyone who has installed the plugin gets it on their next update.
